# CareerMoney365 Blog System (GitHub Pages)

This repo contains a simple blog system that works on GitHub Pages without any backend.

## How to add a new post

1. Create a new file inside `/posts/`
   Example: `my-new-post.html`

2. Add your post HTML content in that file.

3. Open `/posts/posts.json` and add a new object.

4. Commit changes.

Now it will automatically show on Home and Blog pages.

## Pages
- Home: `index.html`
- All posts: `blog.html`
- Single post: `post.html?id=POST_ID`
